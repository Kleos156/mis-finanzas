Para sincronizar datos entre dispositivos necesitas configurar Firebase una sola vez. Es gratis y tarda unos minutos.

1Ve a console.firebase.google.com y crea un proyecto nuevo.
2En el menú lateral → Compilación → Realtime Database → Crear base de datos (modo de prueba).
3Ve a Configuración del proyecto ⚙ → Tus apps → icono Web </> → Registrar app.
4Copia los valores apiKey, databaseURL y projectId y pégalos aquí abajo.
Por favor completa todos los campos.
